## How to Use

1. Open the `exercises.md` for the module you're working on.
2. Try each exercise yourself — don't peek at answers yet!
3. When you're stuck (really stuck, not just a little stuck), open `answers.md`.
4. Type the code yourself, don't copy-paste. Your hands need to remember.
