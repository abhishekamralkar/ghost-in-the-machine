## Your Summer Quest

This summer you're not just learning — you're leveling up.
By the end of 10 weeks you will have **built a real AI-powered device** with a screen, a button, and a tiny computer the size of a credit card. Real tech. Real code. No training wheels.

## The Rules of the Game

- **Ask questions.** No question is dumb. Ever.
- **Break things on purpose.** Seriously — breaking things is how you learn to fix them.
- **Type everything yourself.** No copy-paste. Your fingers need to remember.
- **Celebrate every small win.** Getting `Hello, World!` to print IS a victory — it was for every real programmer too.
- **Collect the badges.** Each module has secret challenges with badges. Try to earn them all!

---

## Tools You'll Need

- A computer running Linux (or a Raspberry Pi)
- A terminal / command line
- Python 3 (already on most Linux systems)
- **VS Code** — the code editor pros use at Google and Netflix (`sudo apt install code`)
- [Ollama](https://ollama.com) — runs AI right on your computer, no internet needed
- Raspberry Pi 4 (for the final project)
- A small OLED display (SSD1306)
