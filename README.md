## What You Will Learn This Summer

| Week | Topic | Cool Thing You'll Build |
|------|-------|------------------------|
| 0    | How Computers Work | Understand CPU, RAM, storage, and the OS |
| 1–2  | Linux + Git | Navigate like a hacker, write shell scripts, track your work with git |

## House Rules for the Internship

- **Ask questions.** No question is dumb.
- **Break things on purpose.** That's how you learn to fix them.
- **Type everything yourself.** No copy-paste — your fingers need to remember.
- **Celebrate every small win.** Getting `Hello, World!` to print is a real victory.

---

## Tools You'll Need

- A computer running Linux (or a Raspberry Pi) — see Module 0 for what these are
- A terminal / command line
- Python 3 (already on most Linux systems)
- **VS Code** — code editor for Python, shell scripts, and everything else (`sudo apt install code`)
- [Ollama](https://ollama.com) — free, runs AI locally on your computer
- Raspberry Pi 4 or 5 (for the final project)
- A small OLED or LCD display (SSD1306 or similar)
